# Legacy CancerPPIr implementation

`cancerppir_legacy.R` is an unchanged copy of `cancerppir.R` created
before the planned refactoring of the CancerPPIr codebase.

It is retained as a reference for:

- comparison of analytical results;
- regression testing;
- verification of code changes;
- recovery of the original workflow behavior.

The file should remain unchanged during refactoring.
